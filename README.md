## Demos

### 1. External Inventory Sync

**File:** [`external-inventory-sync.html`](./external-inventory-sync.html)

A hi-fi UI demo of an external inventory synchronization dashboard.

**Features:**
- **Overview Tab** — Displays the latest sync batch info (Batch No., Sync Time, Total SKUs synced) and inventory variance cards. Clicking a card value jumps to the Specifics tab with the filter pre-filled.
- **Specifics Tab** — A filterable data table showing SKU-level sync records with:
  - Filters: Sync Batch No., Part No., Part Name, Sync Time (date range picker), External Supplier
  - Columns: Sync Batch No., Part No., Part Name, External Supplier, Available Qty (Supplier), Available Qty (ERP), Discrepancy (ERP), Total (ERP), Sync Time
  - Column visibility toggle modal
  - CSV export button
  - Pagination with configurable page size (10 / 20 / 50)
- **Responsive layout** — adapts to tablet and mobile screen sizes
- **Mock data** — 20 winter tyre spare part SKUs across 3 sync batches (Nokian supplier)

**Generated by:** AI (Gemini)
